# EVCS-and-RES-Dataset-of-5-typical-days-20260506
This dataset supports the DSSR research. There are five typical days with different EVCSs datasets and some RES informations on the IEEE-33 and IEEE-141 bus system.

The Appendix of this paper is also presented here:

1) Appendix A includes basic conceptions of DSSR mentioned in this paper, and its geometric analysis categories; 
2) Appendix B covers extra simulation results of different EVCS that wasn't cover in this paper.
